
# React 19 Calculator

Calculadora hecha con React 19 para probar la beta y el compilador.

## Correr localmente

Clona el proyecto

```bash
  git clone https://github.com/Facug03/react-calculator.git
```

Movete al directorio del proyecto

```bash
  cd react-calculator
```

Instala las dependencias

```bash
  npm install
```

Inicia el proyecto

```bash
  npm run dev
```
